# GetYoutubePlayCount

YouTube 動画の再生数を定期取得し、Google スプレッドシートに記録・可視化する Google Apps Script です。

## 機能

- **定期記録** — トリガーで `main` を呼び出すたびに全動画の再生数をシートに追記
- **個別グラフ** — 動画ごとのシートに再生数推移の折れ線グラフを自動生成
- **比較シート** — 全動画の再生数推移を1シートに集約し、一覧グラフを生成
- **データ間引き** — 古いデータを自動間引きしてシートの肥大化を抑制
- **増加量サマリー** — 各動画シートに1時間・1日・1週間・1ヶ月・1年ごとの増加量を表示

## スプレッドシートの構成

```
スプレッドシート
├── 再生数比較          # 全動画の比較シート（自動生成）
├── <動画タイトルA>     # 動画ごとのシート（自動生成）
├── <動画タイトルB>
└── ...
```

### 動画シートの構造

| 行 | 内容 |
|----|------|
| A1 | 動画タイトル（表示用） |
| A2 | 投稿日時（白文字・グラフ起点用） |
| A3:B3 | ヘッダー（日時 / 再生数） |
| A4以降 | 取得日時と再生数のレコード |

### 比較シートの構造

| 列 | 内容 |
|----|------|
| A | 動画名 |
| B | 最新再生数（太字） |
| C | 経過日数（投稿日からの日数） |
| D | 24時間増加数 |
| E | 7日増加数 |
| F | 30日増加数 |

グラフは比較シートのテーブル下に2種類配置されます：
- **絶対日時グラフ** — 全動画の再生数推移（実際の日付が横軸）
- **投稿日起点グラフ** — 投稿日からの経過日数が横軸（動画間の初速を比較しやすい）

## セットアップ

### 1. スプレッドシートを作成

Google スプレッドシートを新規作成し、「拡張機能 → Apps Script」を開く。

### 2. YouTube Data API を有効化

GAS エディタで「サービス → YouTube Data API v3」を追加する。

### 3. コードをデプロイ

```bash
# 依存パッケージのインストール（型定義のみ・実行には不要）
npm install

# GAS にプッシュ
npm run push
```

### 4. トリガーを設定

GAS エディタで「トリガー → トリガーを追加」から以下の2つを設定する。

| 関数 | 種別 | 頻度 | 用途 |
|------|------|------|------|
| `main` | 時間主導型 | 1時間ごと | 再生数の取得・記録 |
| `updateAllCharts` | 時間主導型 | 6時間ごと（推奨） | グラフ・比較シート更新 |

> **なぜ2つに分けるのか**
> GAS の実行時間上限は6分。グラフ生成は重いため、データ収集と分離することでタイムアウトを回避している。

## 設定

`main.gs` 冒頭の `CONFIG` を編集する。

```js
const CONFIG = {
  // 対象プレイリストの ID（不要な場合は空文字）
  PLAYLIST_ID: 'PLxxx...',

  // プレイリスト外で個別追加したい動画 ID
  EXTRA_VIDEO_IDS: ['動画ID1', '動画ID2'],

  // 比較シートの名前
  COMP_SHEET_NAME: '再生数比較',

  // 削除・リセット対象から除外するシート名
  PRESERVE_SHEET_NAMES: ['再生数比較', 'シート1'],

  // 比較グラフのサイズ（ピクセル）
  CHART: {
    WIDTH: 2210,
    HEIGHT: 850,
  },

  // データ間引きの設定
  SAMPLING: {
    MIN_ROWS_TO_SAMPLE: 10,  // この行数以下は間引きしない
    RULES: [
      { maxDays: 30,       keepEveryHours: null }, // 30日以内: 全件保持
      { maxDays: 90,       keepEveryHours: 6    }, // ~90日: 6時間ごと
      { maxDays: 180,      keepEveryHours: 12   }, // ~180日: 12時間ごと
      { maxDays: 365,      keepEveryHours: 24   }, // ~365日: 1日ごと
      { maxDays: Infinity, keepEveryHours: 168  }, // 365日超: 週1
    ],
  },
};
```

## 手動実行できる関数

| 関数 | 用途 |
|------|------|
| `main` | 再生数の取得・記録（グラフ更新なし） |
| `updateAllCharts` | グラフ・比較シート・シート並び替えを更新（データ変更なし） |
| `rebuildComparisonSheet` | 比較シートとグラフだけ再生成（動画データは変更しない） |
| `resetSheets` | 動画シートをすべて削除してリセット（⚠️ データ消去） |
